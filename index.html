Simply Responsive Content

Simply Responsive Content is a small TYPO3 extension providing a 'Responsive Settings' tab on all tt_content elements.

Upon selecting one or more of the responsive visibility settings the respective classes will be added to the normal classes on a content element.

This extension is intended to be used with ZURB's Foundation framework.

What it doesn't do...

  1. it doesn't create your website for you
  2. it doesn't clear you from the duty of mastering TypoScript
  3. it doesn't install ZURB Foundation for you (you should have made your own templates and include Foundation on your own!)

What it does do...

  1. Provides a tab on all content elements where editors can select the different visibility classes for that element. On rendering the selected classes will then be added to the default class wrapping for that element. That's all. Nothing more. Nothing less.

How to use...

  1. Install the extension like any other TYPO3 extension.
  2. Include the static templates.
  3. Et voila: you'll see the responsive settings tab there, working and all.

Running into issues?

Please check first if nothing is actually overriding the tt_content.stdWrap.innerWrap.cObject.default.20 section in your TypoScript.

The classes are added where the default csc classes are also loaded. If you have changed this, you can modify the TypoScript to fit your needs.

Really running into issues?

Either fix it and do a pull request on Github or create an issue there...
